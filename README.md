This is a full-stack e-commerce prototype developed during my internship at TeamLease. The project was built using Django for the backend and React for the frontend, providing a seamless and responsive shopping experience.

📌 Key Features
✅ Product Management: Create, read, update, and delete (CRUD) operations for products and categories.

🛍️ Shopping Cart: Add/remove items, update quantities, and manage cart state with validations.

👥 User Authentication: Secure login and registration using OAuth2.

🖼️ Image Uploads: Reliable image handling with validations and storage support.

📊 Database Schema Design: Custom PostgreSQL schemas for products, categories, carts, and users.

🔒 Data Integrity: Robust handling of null values, field constraints, and input validation.
